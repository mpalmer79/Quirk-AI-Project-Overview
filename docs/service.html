Quirk AI for Service

Status: In Development

Mobile-first tools designed for service advisors and customers, with native Android and iPhone apps to modernize check-ins, approvals, parts lookups, and follow-ups — powered by Quirk AI.

Open Web Preview Join Beta (iOS/Android) — Coming Soon

What it does

  • Schedule & reminders
  • Touchless check-in
  • Service status updates
  • GM parts requests
  • Loyalty & offers
  • Sight-Unseen trade-in

How it works

  1. Pick a time that fits your day.
  2. Check in on arrival — touchless option available.
  3. Track status, approve work, and get notified for pickup.

Built for Quirk Chevrolet

Native iPhone & Android experience with a lightweight web preview. Built with React + Vite, Tailwind, and shadcn/ui. Secure messaging via Netlify Functions. No payment info stored client-side.

Roadmap

  • Now: Scheduling, parts, inspection overview
  • Next: VIN scan, approvals, loyalty integration
  • Planned: Live service status milestones, push notifications
