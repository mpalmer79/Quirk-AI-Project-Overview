What it does

Tier-1 call handling with tool-driven actions: create CRM tickets, hold service slots, quote trades, and check order status. Short, professional voice style; graceful escalation when needed.

Where calls go

  • Sales → Lead + Activity (+ optional Appointment)
  • Service → Vehicle link (VIN/YMMT) + Service Appointment
  • Parts → Quote lines + optional Pick-up/Ship setup
