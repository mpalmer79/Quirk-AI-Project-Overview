Creatio integrates with OpenAI, but there is no direct, native integration between Creatio and PBS Dealer Systems. Integration with PBS would require using Creatio's flexible API and a third-party connector or custom development.

Creatio and OpenAI integration

Creatio features a built-in AI copilot that uses OpenAI's large language models (LLMs). It can also connect to other services. This integration is available out-of-the-box and offers these features:

  • Built-in capabilities: Creatio Copilot is LLM-agnostic. It supports OpenAI's GPT models, as well as Anthropic Claude and Google Gemini, to assist users. A ChatGPT connector is also available through the Creatio Marketplace.
  • Use cases: The AI assists with text generation for emails, chats, and social posts. It can also act as a virtual assistant by referencing knowledge base articles to suggest solutions for customer service cases.
  • Developer tools: For no-code developers, Creatio Copilot can assist with the app-development process, such as generating app templates based on a description.
  • Custom AI skills: Custom AI skills can be created to perform more complex, multi-step tasks with a single prompt.

Creatio and PBS Dealer Systems integration

While there is no pre-built connector, integrating Creatio with a specific Dealer Management System (DMS) like PBS is possible using several methods.

  • Open API: Creatio offers REST API support and OData protocol, allowing custom integrations with third-party systems.
  • Integration Platform as a Service (iPaaS): Partners like Cyclr and Tray.io can enable connections with a wider range of applications.
  • Custom connectors: Businesses can develop their own connectors using Creatio's open platform to exchange data with PBS Systems.
  • Integration partners: Specialized partners may build a custom solution for specific needs.

Integrating the three systems

The most effective way to combine all three systems is to use Creatio as the central hub:

  • Data exchange: Implement a custom integration between Creatio and PBS Dealer Systems to sync dealership data (such as sales, customer, and vehicle information) into Creatio.
  • AI-powered processes: Use Creatio's native OpenAI integration to analyze, augment, and automate business processes using that dealership data. For example:
    • Use AI to generate personalized follow-up emails for new sales leads based on data imported from PBS.
    • Leverage AI to create summaries of customer service chats to log in Creatio, referencing case information from the PBS integration.

Salesforce CRM is a cloud-based platform that unifies sales, service, marketing, and commerce on a single integrated platform with AI-driven insights and automation.

Expand Collapse

Salesforce can be integrated with OpenAI and PBS Systems, but the process and capabilities differ for each. Integration with OpenAI is a widely supported native and API-driven process, while integrating with PBS Systems, a Dealer Management System (DMS), requires custom development because the two products have competing functionalities.

Integration with OpenAI

Salesforce and OpenAI can be integrated in a variety of ways to embed AI capabilities directly into CRM workflows.

  • Native AI features: Salesforce’s own AI, Einstein, is enhanced by a strategic partnership with OpenAI to provide generative AI features.
  • API integration: Developers can use Apex to connect Salesforce to OpenAI’s APIs for custom use cases like intelligent lead scoring, automated summaries, and advanced customer service.
  • Third-party connectors: Platforms like n8n.io and Relay.app offer low-code solutions to connect Salesforce and OpenAI through pre-built templates and workflows.

Integration with PBS Systems DMS

Integrating Salesforce with PBS Systems is a more complex undertaking, as PBS is a comprehensive, standalone DMS that also contains its own CRM functions.

  • Custom API development: Integration is primarily achieved through custom API development. PBS Systems provides an API toolkit for partners to build integrations with its v10 DMS.
  • No out-of-the-box solution: Since Salesforce offers its own competing Automotive Cloud, there is no simple, “plug-and-play” connector for PBS Systems on the AppExchange.
  • Specialized consultants: Dealerships often work with specialized integrators or consulting firms, who use advanced APIs and platforms like MuleSoft to build a bi-directional data sync between the two systems.
  • Data synchronization: Key to this integration is mapping data fields and building a synchronization strategy to avoid duplicates and ensure consistency for customer, vehicle, and inventory information.
